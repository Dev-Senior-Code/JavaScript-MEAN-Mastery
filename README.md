# JavaScript MEAN Mastery - Tutorías Andrés Hernández

Este repositorio está diseñado para centralizar los materiales, prácticas y recursos de aprendizaje del programa **JavaScript MEAN Mastery**, guiado por Andrés Hernández.  

El objetivo es formar desarrolladores con un dominio integral del **stack MEAN (MongoDB, Express.js, Angular, Node.js)**, comenzando por los fundamentos del desarrollo web moderno.



##  Contenido del Programa

**Objetivo:**  
Dominar los **fundamentos de JavaScript, HTML y CSS**, y sentar las bases del desarrollo web.
